# Lextry Study

Lextry Study es una plataforma universitaria inteligente diseñada para ayudar a los estudiantes a organizar, planificar y optimizar su vida académica en un solo lugar.

## Funcionalidades principales

* Planificación semestral inteligente
* Seguimiento de notas y rendimiento académico
* Control de asistencia y fallas
* Organización de entregas, parciales y horarios
* Currículo digital con prerrequisitos
* Herramientas de estudio impulsadas por IA
* Focus sessions y temporizador Pomodoro
* Compatibilidad con múltiples universidades

## Tecnologías

* Next.js
* TypeScript
* Tailwind CSS
* shadcn/ui
* Framer Motion

## Desarrollo

Este proyecto está siendo desarrollado con apoyo de herramientas de inteligencia artificial como ChatGPT y Manus AI para procesos de diseño, arquitectura y desarrollo frontend.

## Objetivo

Crear una plataforma moderna, visualmente atractiva e intuitiva que centralice todas las herramientas que un estudiante universitario necesita para gestionar su experiencia académica de manera más eficiente.
